## Movie Correlation Analysis (Python)

This project explores relationships between different movie features such as budget, gross revenue, votes, ratings, and runtime using correlation analysis and visualization in Python.

The goal is to identify which variables are most strongly associated with movie financial success and audience engagement.

---

## Objectives

- Clean and prepare a movies dataset for analysis  
- Convert categorical variables into numerical form  
- Perform correlation analysis using Pearson correlation  
- Visualize relationships using heatmaps and scatter plots  
- Extract insights about what drives movie revenue  

---

## Tools & Libraries

- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  

---

## Key Findings

- Budget and Gross Revenue show strong positive correlation  
- Votes and Gross Revenue show strong positive correlation  
- Higher audience engagement tends to be associated with higher revenue  
- Most categorical features have weak direct correlation with revenue
